# Ecofin

An open-source ecosystem of financial literacy tools, knowledge, and infrastructure.
We build in public. Everything we learn about money, markets, and economics — we document, open-source, and share.

## Projects

- [Ecofin Blog](https://blog.ecofin.uz/) — Financial literacy articles in Uzbek, Russian, and English. Focused on practical knowledge: loans, interest, inflation, markets.
- Ecofin Platform — A personal finance tracker and budgeting tool. Currently in development.
- [ecofin-blog](https://github.com/ecofinuz/ecofin-blog) — The open-source codebase behind the blog. Built with Astro v5, multi-language, interactive components.
- Ecofin Django — An open-source Django project for financial data and tooling. In progress.

## Why Ecofin?

Financial literacy in Uzbekistan and the broader Central Asian region is underdeveloped. Most quality content exists in English, behind paywalls, or buried in academic language.
Ecofin is an attempt to fix that — starting with a blog written in Uzbek, Russian, and English, and expanding into tools that people can actually use.

## Stack

- **Blog** — Astro v6, MDX, Netlify
- **Platform** — Django 6.0, PostgreSQL
- **Languages** — TypeScript, Python
